# 示例图片说明

这个目录用于存放 QQ Bot 帮助菜单生成工具的示例图片。

## 如何添加示例

1. **使用本工具生成图片**:
   - 启动应用 (http://localhost:5000)
   - 在网页中配置你的 QQ Bot 信息和菜单内容
   - 点击"下载图片"按钮获得 PNG 格式的菜单图片

2. **保存示例图片**:
   - 将生成的图片重命名为 `example.png`
   - 保存到本目录中 (docs/examples/)

3. **图片要求**:
   - 格式: PNG
   - 推荐宽度: 400-800px
   - 用途: 在 README 中展示工具效果
